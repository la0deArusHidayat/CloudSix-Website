Tentang Tim cc-06 Bangkit
Cloud Computing 2024

Kami adalah tim cc-06, salah satu tim peserta program Bangkit Cloud Computing 2024 Batch 1 dari Kampus Merdeka. Terdiri dari individu-individu dengan semangat tinggi dalam mempelajari cloud computing, kami bersatu dengan visi untuk mengembangkan keterampilan dan menciptakan solusi inovatif menggunakan teknologi cloud.

Dalam program ini, kami akan mempelajari konsep dasar cloud computing, mengikuti pelatihan intensif, dan mengerjakan proyek nyata dengan menggunakan platform cloud Google Cloud Platform (GCP). Melalui kolaborasi tim yang solid, kami bertekad mencapai hasil terbaik dalam proyek kami.
